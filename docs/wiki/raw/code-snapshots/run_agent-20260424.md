## Invariants enforced in this module

- `_install_safe_stdio()` is called unconditionally at the start of `AIAgent.__init__` and `run_conversation()`.
- Iteration budget is a single shared counter across a delegation tree (`iteration_budget or IterationBudget(max_iterations)`); subagents never get a fresh budget.
- Built-in agent-level tools (`todo`, `memory`, `session_search`, `delegate_task`, `clarify`) are intercepted in `_invoke_tool` and never fall through to `handle_function_call` / registry dispatch.
- Subagent delegation state: `self._delegate_depth` (0 = top-level, incremented for children) and `self._active_children` (list of running children, protected by `_active_children_lock`).
- Memory provider tool names conflicting with already-registered tool names are skipped in the injection loop to avoid 400 errors on providers (e.g. Xiaomi MiMo via Nous Portal) that enforce unique function names.
- Plugin pre-tool-call hooks can block tool execution by returning a block message from `get_pre_tool_call_block_message(...)`; both `_invoke_tool` (line 7258) and `model_tools.handle_function_call` check this, with a `skip_pre_tool_call_hook=True` bypass to prevent double-firing when the caller has already checked.
